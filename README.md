# 🧱 Tetris Retro – Projet Python

Un jeu **Tetris rétro** développé en Python à l’aide de la bibliothèque `pygame`.
Ce projet s’inscrit dans un parcours d’apprentissage rigoureux et passionné visant à maîtriser Python en profondeur à travers des projets concrets et fun.

---

## 🎮 Objectifs pédagogiques

- Maîtriser la **programmation événementielle avec Pygame**
- Travailler avec des **grilles logiques et matrices**
- Gérer les **collisions, rotations, suppression de lignes**
- Structurer un projet Python comme un vrai développeur
- Approfondir les **structures de données**, les **algorithmes** et la **gestion du temps**
